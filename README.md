# Chess Game

This will allow you to play chess.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Open the C++ project in your IDE.
2. Run the Build function of your IDE.
3. Click run, either the debugger or just run the program.

Instructions for using the software:

1. White is the Capital letters, you move by first saying the grid square where your piece at and then the grid square where you want it to go. For example, b2b4.
2. White goes first.
3. The standard rules of Chess apply. I might have missed some.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* <iostream>
* <string>
* You must have the c++ language downloaded.
